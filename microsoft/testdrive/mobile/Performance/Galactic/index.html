This demo shows a 3D scene rendered using Javascript and HTML5 Canvas. The scene depicts part of our galaxy, containing the sun and several planets. Each planet orbits the sun, and Earth's moon orbits Earth. You can change the viewpoint by pressing the Q key. The +/- keys speed up and slow down the simulation, and the R key resets everything to default values. Keys 1-5 toggle layers in the scene, hiding and showing different objects.
